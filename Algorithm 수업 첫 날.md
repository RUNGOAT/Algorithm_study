# 일의 자리 수 추출

```python
num = 45678

while num > 0
	lst.append(num % 10)	# num의 일의 자리 수 추출
	num //= 10	# 일의 자리 수 추출 후 삭제

>>> lst
# [8, 7, 6, 5, 4]
```





메모리를 늘리더라도 (배열 끝에 공간 추가)

비교 연산을 줄이는 것이 단순해질 수 있다.
